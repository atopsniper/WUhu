# ⚡️ WUhu - Next-Gen EdgeTunnel
**高性能、高隐蔽性的 Cloudflare Edge 计算代理平台**

WUhu 是基于 Cloudflare Workers 构建的顶级代理解决方案，集成了深度混淆、动态防探测与实时监控中心。

### ✨ 特性
* **黑客级伪装：** 流量特征深度混淆，有效对抗 DPI。
* **实时监控：** 内置极客风指挥中心，监控节点运行状态。
* **极速响应：** 自动预热机制，告别冷启动等待。
* **安全防御：** 动态路由伪装，拒绝非法扫描。

### 🚀 部署步骤
1. **Fork 本仓库** 到你的 GitHub。
2. 在 Cloudflare 面板创建一个新的 **Workers**。
3. 将 `_worker.js` 代码粘贴进去。
4. 在变量 (Variables) 中设置你的 `UUID` 和 `PASSWORD`。
5. 绑定域名，即可开始使用！
